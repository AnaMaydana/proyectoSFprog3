Proyecto Grupal: Sistema de Gestión de Hotel

Realizado por el Grupo 3

Descripción del Proyecto

Este proyecto es un sistema de gestión para hoteles, desarrollado en C++ con interfaz interactiva. Permite registrar clientes, gestionar reservas, administrar habitaciones, controlar membresías y generar reportes diarios. Está diseñado para ser utilizado por el personal del hotel y mejorar la eficiencia en la gestión de operaciones diarias.
